# NostalBrain

## Missão

O NostalBrain é o cérebro do NostalCore.

Seu objetivo é reconhecer automaticamente qualquer jogo importado.

O usuário nunca deve precisar:

- escolher console

- escolher capa

- escolher banner

- organizar biblioteca

Tudo acontece automaticamente.
