# NostalCore

## Logo Oficial

A identidade visual oficial do NostalCore é composta por um símbolo baseado na letra "N" dentro de uma moldura arredondada.

Ela representa:

- Nostalgia
- Núcleo (Core)
- Organização
- Tecnologia
